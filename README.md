# Shivam Pandey Task Management App

A full-stack task management application with
web dashboard and Android app.

## Tech Stack
- Node.js
- MongoDB
- React / Next.js
- Android (WebView)
- Firebase Notifications

## Features
- Secure login
- Task priority & deadlines
- Reminder notifications
- Team task assignment

## Status
Backend + Frontend + Android app architecture ready.
